### Задание 1
Реализуйте функцию `stringCopy`
```cpp
char* stringCopy(const char*);
```
```cpp

char * newString = stringCopy(p);
p[0] = 'S';
std::cout << p << std::endl; // Some string
std::cout << newString << std::endl; // some string
```
